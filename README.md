# highway-pilot-em
An automated design co-pilot for highway electromechanical engineering: CAD layout, BOM statistics, and topology generation. 高速公路机电工程设计的“自动驾驶”助手：支持沿线布点、工程量统计及供电/网络拓扑图自动生成。 Un copilote de conception automatisé pour l'ingénierie électromécanique autoroutière : implantation CAO, statistiques de nomenclature (BOM) et génération de topologie.
